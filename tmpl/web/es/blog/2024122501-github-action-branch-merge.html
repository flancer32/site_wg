Nuevo proyecto: GitHub Action para fusionar ramas con exclusión de archivos

Fecha de publicación: 2024-12-25

He completado un proyecto para crear un GitHub Action que fusiona automáticamente una rama con otra excluyendo ciertos archivos. Luego, la rama actualizada se envía de vuelta al repositorio.

Durante la implementación, descubrí que un Personal Access Token (PAT) no es suficiente para enviar cambios de forma segura — fue necesario configurar la autenticación SSH para una interacción confiable con el repositorio.

Este proyecto ha sido una gran experiencia práctica. Si necesitas configurar CI/CD con GitHub Actions, te invito a ver mi Gig en Fiverr.

¡Feliz codificación!
