TeqCMS: CMS del lado del servidor con plantillas Nunjucks

Fecha de publicación: 2025-05-29

Se ha publicado un conjunto de paquetes para crear un CMS con generación de páginas del lado del servidor usando plantillas Nunjucks. El nuevo sistema está construido sobre la arquitectura TeqFW e incluye módulos para servidor web, renderizado de plantillas y gestión de contenido:

  • @flancer32/teq-web — servidor web modular con soporte para HTTP/2
  • @flancer32/teq-tmpl — renderizador universal para Nunjucks y Mustache
  • @flancer32/teq-cms — CMS orientado a archivos con soporte para localización
  • @flancer32/teq-cms-demo — sitio web de demostración

Todos los paquetes utilizan @teqfw/di para inyección de dependencias, están diseñados como plugins y se pueden combinar fácilmente. La demostración muestra cómo crear un sitio multilingüe sin base de datos con SSR completo.

El sistema está adaptado para alojamiento estático, traducción automática y funcionamiento completo sin bundlers o rastreadores de terceros.

Ejemplo de sitio en funcionamiento: cms.teqfw.com
