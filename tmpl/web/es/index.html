Quién soy

Soy Alex Gusev, desarrollador web independiente en Letonia. Mi trabajo se centra en la arquitectura JavaScript, la infraestructura propia y los procesos guiados por agentes. Uso estas herramientas en mis propios productos: Tequila Framework, GitHub Flows, sitios web generados con IA y basados en TeqFW, y este sitio.

Productos clave

Tequila Framework

Mi principal producto de ingeniería: una plataforma JavaScript y una base arquitectónica para sistemas que deben conservar una estructura clara a medida que crecen.
