Sobre mí

Soy Alex Gusev, desarrollador web independiente en Letonia. Me centro en la arquitectura JavaScript, la infraestructura autohospedada y los flujos de trabajo guiados por agentes. Todo eso lo aplico en mis propios productos: Tequila Framework, GitHub Flows, sitios web generados con IA sobre TeqFW y este mismo sitio.

Productos clave

Tequila Framework

Mi producto técnico principal: una plataforma JavaScript y una base arquitectónica para sistemas que necesitan seguir siendo claros y ordenados a medida que crecen.

Filosofía de Tequila Framework

GitHub Flows

Mi oferta principal ahora mismo: una capa de proceso construida sobre eventos de GitHub y sobre el tratamiento de incidencias, PR, etiquetas y comentarios mediante agentes dentro de la infraestructura del cliente. La aplicación anfitriona es secundaria; el producto real es el proceso.
