ADSM / GitHub Flows

Desarrollo controlado con agentes de IA para aplicaciones web

Diseño procesos basados en GitHub donde un cambio de producto avanza desde una solicitud inicial hasta la ejecución por un agente, la propuesta de cambio, la validación y la publicación bajo control documentado.

Ver demostración pública Hablar de un piloto

Prueba pública: solicitud en GitHub → agente de IA → página publicada

En TeqFW Demo Pages, cualquier usuario de GitHub puede abrir una solicitud acotada para pedir una página de demostración. Si la solicitud encaja en los límites documentados, GitHub Flows activa al agente, el agente crea una página dentro de esos límites y el resultado se publica como artefacto visible.

No es solo una pieza de portafolio. Es una prueba pública de que la evolución controlada de un producto puede apoyarse en solicitudes de GitHub, restricciones documentadas, ejecución por agentes, revisión y publicación.

Qué demuestra esto

  • La intención del usuario puede entrar en el producto a través de solicitudes en GitHub.
  • Los agentes de IA pueden trabajar dentro de restricciones documentadas.
  • GitHub aporta trazabilidad mediante solicitudes, etiquetas, comentarios, propuestas de cambio y evidencia de validación.
  • La persona responsable del producto mantiene el control sobre cambios permitidos, puntos de revisión y publicación.

Pila metodológica

ADSM es el método.

Agent Driven Software Management define cómo el contexto, la documentación, las restricciones, la revisión y la evidencia mantienen bajo control el desarrollo impulsado por agentes.

GitHub Flows es la capa de ejecución.

Conecta eventos de GitHub, solicitudes, etiquetas, propuestas de cambio, ejecución de agentes en servidor, comentarios y validación dentro de un flujo de trabajo controlado.

Tequila Framework es la plataforma de aplicación.

Proporciona una base en JavaScript para aplicaciones web pensadas para trabajar con agentes: DI, estructura modular, servidor web, configuración, registro, capa de persistencia y componentes relacionados.
