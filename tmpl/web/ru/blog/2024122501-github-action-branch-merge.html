Новый проект: GitHub Action для слияния веток с исключением файлов

Дата публикации: 2024-12-25

Завершил проект по созданию GitHub Action, который автоматически сливает одну ветку в другую с исключением определённых файлов. После этого обновлённая ветка отправляется обратно в репозиторий.

В процессе реализации оказалось, что для безопасной отправки изменений недостаточно Personal Access Token (PAT) — потребовалась настройка SSH-аутентификации для надёжного взаимодействия с репозиторием.
