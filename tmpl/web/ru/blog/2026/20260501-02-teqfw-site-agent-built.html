teqfw.com собран Codex-агентом

1 мая 2026

Я обновил свой сайт teqfw.com: главную страницу и связанные с ней материалы.

Сайт собран на базе TeqFW. Код, тексты, дизайн и разметка страницы были полностью подготовлены Codex-агентом.

Работа над сайтом велась вместе с Codex-агентом по моей методологии ADSM, Agent-Driven Software Management.

Объёмы материалов для этого обновления такие:

440Кб - контекст проекта, то есть документация;
512Кб - спецификации для TeqFW и SSR-сайта;
77Кб - код приложения для обработки шаблонов через Nunjucks;
80Кб - текст шаблонов, то есть разметка и страницы;
160Кб - статика, включая изображения.

Вывод здесь практический: при таком способе работы документация должна заметно превосходить код по объёму. Тогда агенту проще удерживать рамку и последовательно собирать результат.
