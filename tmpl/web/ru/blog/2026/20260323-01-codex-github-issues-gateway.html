Шлюз между GitHub Issues и Codex-агентом

23 марта 2026

Шлюз между GitHub Issues и Codex-агентом был собран как proof of concept, но уже с рабочим контуром. Он подключался к GitHub от имени пользователя adsm-agent, а в него были заведены три репозитория: сам шлюз, сайт teqfw.com и транслятор постов в Telegram на английский и испанский языки. Профиль агента: github.com/adsm-agent.

Через этот контур было создано несколько issues, которые затем разобрал агент. В части задач результат получился точным и быстрым. В части задач агент пока не справился с контекстом и ограничениями так, как требовалось. Общая картина вышла понятной: схема работает, но ей ещё нужно довести поведение до устойчивого рабочего состояния.

В практическом плане это уже даёт удобный маршрут для постановки задач агенту через GitHub. Следующий шаг связан с доводкой правил, отладкой входного потока и более ровной обработкой задач разного типа.

Отдельная задача, которую агент уже обработал: teqfw/di#44 .
