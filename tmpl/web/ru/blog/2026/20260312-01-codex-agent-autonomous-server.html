Запуск Codex-агента в автономном режиме на сервере

12 марта 2026

12 марта Codex-агент был поставлен в автономный режим на один из виртуальных серверов в США и подключён к OpenAI через API-ключ. С третьей попытки удалось добиться простого результата: агент сам создал в текущем каталоге текстовый файл с приветствием Hello World!.

Самое забавное было в том, что агент постоянно пытался уйти в интерактивный режим. На создание одного маленького файла у него ушло около 10 тысяч токенов, то есть примерно 3 цента. После этого только укрепилось ощущение, что для постоянной работы Codex по подписке заметно выгоднее.

Подписка за 20 евро в месяц позволяет работать очень долго. По крайней мере, за полгода ни разу не приходилось упираться в лимиты, хотя через Codex и не кодилось что-то по-настоящему большое.

Дальше видится вполне практичный сценарий: поставить небольшое веб-приложение для приёма GitHub webhook'ов, например при создании issue, и через него запускать агента. Такой агент сможет анализировать текст задачи, передавать её в работу или сразу отклонять. При нормальном финансировании из этого уже можно собрать маленькое автоматическое агентство разработки. По скорости оно точно не будет хуже ручной обработки задач в Magento. Но токены, конечно, будет сжигать заметно.
