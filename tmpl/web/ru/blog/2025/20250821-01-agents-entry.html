AGENTS.md как точка входа

Дата публикации: 2025-08-21

Долго думал, какой файл сделать точкой входа для агентов в своих проектах. Остановился на AGENTS.md. Это оказалось не просто удобное имя, а уже сложившийся формат, который поддерживается многими инструментами и используется в тысячах репозиториев.

По сути, это README для агентов: отдельное место, где фиксируются команды, правила и спецификации. Такой подход разгружает документацию для людей и даёт агентам предсказуемую опору. В ADSM это особенно важно — контекст переносится от проекта к проекту и остаётся понятным.

Подробнее — на agents.md.
