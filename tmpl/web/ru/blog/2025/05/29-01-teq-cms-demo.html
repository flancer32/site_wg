TeqCMS: серверная CMS на шаблонах Nunjucks

Дата публикации: 2025-05-29

Опубликован комплект пакетов для создания CMS с серверной генерацией страниц на шаблонах Nunjucks. Новая система построена по архитектуре TeqFW и включает модули для веб-сервера, рендеринга шаблонов и управления контентом:

  • @flancer32/teq-web — модульный веб-сервер с поддержкой HTTP/2
  • @flancer32/teq-tmpl — универсальный рендерер для Nunjucks и Mustache
  • @flancer32/teq-cms — файл-ориентированная CMS с поддержкой локализации
  • @flancer32/teq-cms-demo — демонстрационный сайт

Все пакеты используют @teqfw/di для внедрения зависимостей, оформлены как плагины и легко комбинируются. Демонстрация показывает, как создать многоязычный сайт без базы данных с полным SSR.

Система адаптирована для статического хостинга, автоматического перевода и полной работы без сборщиков или сторонних трекеров.

Пример работающего сайта: cms.teqfw.com
