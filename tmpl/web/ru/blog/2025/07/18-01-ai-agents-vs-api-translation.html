API и агенты: два подхода к переводу страниц

Дата публикации: 2025-07-18

В TeqCMS переводы страниц изначально выполнялись через API пофайлово. Этот способ прост и надёжен: один файл — один запрос, результат легко встроить в CI/CD.

Но недавно я попробовал другой вариант — использовать AI-агента, который работает сразу с несколькими страницами за одну сессию. Агент может учитывать общий контекст сайта, переводить серию файлов последовательно и даже вносить доработки прямо в процессе.

Оба подхода ценны: API обеспечивает стабильность и предсказуемость, а агенты дают гибкость и позволяют работать в редакторском стиле. Я вижу в этом сочетании практическую пользу и буду развивать направление агентных переводов.
