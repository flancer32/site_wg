# Обработчик переноса маршрутов (redirect-map)

**Path:** `ctx/rules/teqcms/adapter/redirect.md`

## Область ответственности

Обработчик подменяет путь входящего запроса по карте переноса, обеспечивая доступ к страницам, чей адрес изменён.
Подстановка выполняется непосредственно в `req.url` до вызова адаптера.
HTTP-редиректы не используются, статические ресурсы не обрабатываются.

---

## Источник данных

Таблица переноса хранится в:

```text
etc/redirect-map.json
```

Формат:

```json
{
  "/foo.html": "/blog/2025/09/05-01-abdsm-habr.html",
  "/blog/2024/legacy-post.html": "/blog/2025/updated-post.html",
  "/blog/2022/04/07-01-joined-select-knex.html": "/blog/2022/20220407-01-joined-select-knex.html"
}
```

Ключи и значения — абсолютные пути **с расширением `.html`**, без локали.
Файл читается относительно корня приложения, определяемого конфигурацией.

---

## Правила обработки

1. Обработчик применяется только к HTML-маршрутам; статика исключена.

2. Сопоставление выполняется по нормализованному `cleanPath`, предоставленному CMS-хэлпером.

3. Если в карте указан путь вида:

```text
/blog/2022/04/07-01-joined-select-knex.html
```

то подстановка выполняется и для:

- запроса к файлу:
  `/blog/2022/04/07-01-joined-select-knex.html`
- запроса к каталогу:
  `/blog/2022/04/07-01-joined-select-knex/`
  где `cleanPath` равен `/blog/2022/04/07-01-joined-select-knex`

4. При нахождении соответствия:

- выбирается новое значение из JSON,
- поверх него накладывается локаль (если присутствует),
- query-параметры сохраняются,
- итоговый путь записывается в `req.url`.

5. При отсутствии совпадений запрос остаётся неизменным.

---

## Формат результата

Обработчик не возвращает данных.
Он модифицирует `req.url`, после чего управление переходит адаптеру (`getRenderData`).

---

## Инварианты

1. `redirect-map.json` — единственный источник правил переноса.
2. Подстановка выполняется только через изменение `req.url`.
3. HTTP-редиректы не применяются.
4. Статические ресурсы игнорируются.
5. Локаль и query-параметры всегда сохраняются.
