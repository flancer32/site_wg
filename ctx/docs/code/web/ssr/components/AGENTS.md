## Purpose

Defines the documentation branch for reusable browser-facing units delivered by the SSR site shell and page templates.

## Level Map

- `AGENTS.md` — level definition for the component-documentation branch.
- `web-components.md` — catalogue of custom elements delivered by the site.
- `widgets.md` — catalogue of reusable product/UI widgets composed by templates.
- `zoom-img.md` — browser contract for the accessible zoomable-image custom element.

## Level Boundary

Defines:

- Reusable widget identities visible across pages.
- Browser-facing custom-element contracts used by authored templates.
- Relationships between template widgets and technical Web Components.

Does NOT define:

- Product positioning or page-specific commercial copy.
- Backend handlers, persistence, or deployment behavior.
- CSS implementation details or one-off content fragments.
