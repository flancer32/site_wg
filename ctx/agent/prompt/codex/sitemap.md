# Finalization of changes

Можно сделать отчёт по всем выполненным работам.

Собери или обнови `web/sitemap.xml`. Сделай коммит и пуш всех изменений, которые сейчас есть в репозитории - и своих, и чужих.

Если пуш прошёл успешно, то запусти GitHub Action `.github/workflows/publish.yml` для публикации изменений в проде.
