# `NutriLog`: общее описание

## Продукт

`NutriLog` — ИИ-ассистент (кастомизированный GPT-чат) для контроля питания с целью похудания, интегрированный через
действия (actions) с набором инструментов, предоставляемых веб-приложением. Ассистент помогает пользователям
контролировать потребление калорий, даёт свои рекомендации и психологическую поддержку. Продукт является скорее "
_помощником_", чем "_тренером_", и будет полезен для людей, **самостоятельно** борющихся с лишним весом.

## Компоненты продукта

- Кастомизированный GPT-чат (https://chatgpt.com/g/g-FaoraVhde-nutrilog)
- Веб-приложение (https://nutrilog.app.wiredgeese.com/):
- информационно-представительская роль (статические страницы)
- инструменты для GPT-чата (REST API)
- регистрация и авторизация пользователей для чата
- кабинет пользователя (профиль, платежи, выгрузка данных)

## Основной сценарий использования

Полноценный пользователь в свободной форме оповещает ИИ-ассистента о своей цели (например, "_сбросить вес до 85 кг к
такой-то дате_"). Ассистент предлагает стратегию достижения поставленной цели, основываясь на данных о пользователе (
пол, возраст, рост, вес, активность). Например, "_ежедневно нужно потреблять на 200 ккал ниже уровня BMR_".

После того как цель сформулирована, Ассистент помогает пользователю оценить калорийность потребляемых им в течение дня
продуктов и ведёт их учёт. Список потребляемых продуктов и их калорийность сохраняется во внешнем приложении через
инструменты (действия). Ассистент обеспечивает психологическую помощь пользователю в достижении поставленной цели.

Ассистент, по просьбе пользователя, может запросить через инструменты накопленные данные, провести их анализ и дать свои
рекомендации, либо визуализировать движение пользователя к цели.

## Функционал веб-приложения

Веб-приложение обеспечивает следующие услуги:

- информационная поддержка проекта (тексты, медиа, ссылка на внешние ресурсы)
- регистрация новых пользователей
- аутентификация пользователей и восстановление доступа к приложению
- авторизация Ассистента на выполнение действий от имени пользователя
- кабинет пользователя (настройка профиля, просмотр и выгрузка данных)
- инструменты Ассистента (формирование цели, доступ к профилю пользователя, ведение дневника потребления калорий, учёт
изменений веса пользователя)

## Монетизация

Кастомизированный GPT-чат доступен для всех зарегистрированных в OpenAI пользователей. Но для полноценного использования
возможностей GPT-чата (без ограничений на кол-во запросов) пользователю нужно иметь платную подписку на GPT-чат (от $
20/мес). При таких условиях целесообразно вводить небольшую годовую оплату за использование в чате инструментов
веб-приложения (дневник потребления калорий и изменения веса).

**Предложение по стоимости подписки за год:** $40–$50.

## Маркетинг (привлечение пользователей)

Для привлечения пользователей рекомендуется использовать следующее:

1. **Сотрудничество с фитнес-блогерами и влиятельными личностями.** Привлечение популярных тренеров и блогеров в области
здоровья, которые смогут порекомендовать продукт своей аудитории.

2. **Контент-маркетинг.** Создание статей и видеоматериалов, рассказывающих о процессе похудения с помощью
ИИ-ассистента. Поддержка интереса с помощью персонализированных рекомендаций.

3. **Реклама в социальных сетях.** Таргетированная реклама в Facebook, Instagram и TikTok, а также реклама через Google
Ads. Можно также провести рекламные кампании на тематических форумах и группах о здоровье.

4. **Партнёрства с фитнес-центрами и тренерами.** Прямые рекламные предложения от фитнес-центров, тренеров и спортивных
врачей.

Использование вышеописанных стратегий позволит добиться широкого охвата и привлечь первых 5 000–10 000 пользователей за
первый год существования продукта.
