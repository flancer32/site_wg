# Заметки по разработке

В разработке сайта широко используются веб-компоненты, которые находятся в каталоге `./public/comonents/`.

## Базовые разметки (layouts)

Разметка - это astro-файл, который находится в каталоге `./src/layouts/`. Всего используется две базовых разметки:

* `home.astro`: разметка для домашней страницы.
* `base.astro`: разметка для всех остальных страниц.
