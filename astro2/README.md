# Web site for `wiredgeese.com`

The site is [Astro](https://docs.astro.build/en/basics/project-structure/) based app.

The `pages`:

* `./dashboard/` (`./home/`): the homepage with intro and events list.
* `./about/`: the details about the person.
* `./offer/`: the overview of the services.
* `./contact/`: the profiles, addresses and contacts.
* `./blog/{post}`: a single post with an event.
